## Create Database (testdb)

1. ```
   mysql> show databases;
   +--------------------+
   | Database           |
   +--------------------+
   | information_schema |
   | mysql              |
   | performance_schema |
   | sakila             |
   | sys                |
   | world              |
   +--------------------+
   6 rows in set (0.01 sec)
   ```

2. ```
   mysql> create database testdb;
   Query OK, 1 row affected (0.00 sec)
   ```

3. ```
   mysql> show databases;
   +--------------------+
   | Database           |
   +--------------------+
   | information_schema |
   | mysql              |
   | performance_schema |
   | sakila             |
   | sys                |
   | testdb             |
   | world              |
   +--------------------+
   7 rows in set (0.00 sec)
   ```
